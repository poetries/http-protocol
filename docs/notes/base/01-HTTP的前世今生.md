- `HTTP` 协议始于三十年前蒂姆·伯纳斯 - 李的一篇论文
- `HTTP/0.9` 是个简单的文本协议，只能获取文本资源；
- `HTTP/1.0` 确立了大部分现在使用的技术，但它不是正式标准；
- `HTTP/1.1` 是目前互联网上使用最广泛的协议，功能也非常完善；
- `HTTP/2` 基于 Google 的 `SPDY` 协议，注重性能改善，但还未普及；
- `HTTP/3` 基于 Google 的 `QUIC` 协议，是将来的发展方向
